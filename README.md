## Main Goal

The objective is to complete 3 laps in less than 3 minutes. You will be capable?

## Controls

W: Go forward.

S: Go backward.

D: Turn right.

A: Turn left.

R: Respawn car.

SPACE: Respawn in checkpoint.

F1: Debug physics.
